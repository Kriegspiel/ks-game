Supported Kriegspiel rules:

1. Berkeley
2. Berkeley + Any

In plan:

1. Crazykrieg
2. Crazykrieg + Any


### How-to

Install
``` bash
pip install kriegspiel
```

Run tests
``` bash
# clone
git clone https://github.com/Kriegspiel/ks-game.git
# install dependencies
cd ks-game && pip install -r requirements.txt
# run tests
python -m pytest tests/
```
